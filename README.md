# My Personal Resume Website

A multi-page personal resume site built using pure semantic HTML5 (no CSS yet — styling coming in the next phase).

## 🔗 Pages

- `resume.html` — Main page with introduction, hobbies, and links to other pages
- `education.html` — Educational background in table format
- `skills.html` — Technical skills list with nested sub-categories (frameworks per language)

## 🎯 What I Practiced

- Semantic HTML5 tags: `header`, `main`, `section`, `nav`, `footer`
- Proper heading hierarchy (single `h1` per page, `h2` for sections)
- Correct nested list structure (`ol`/`ul` inside `li`)
- Multi-page linking with `<a href>`
- Building tables with `<table>`, `<tr>`, `<th>`, `<td>`

## 🛠️ Tech Used

- HTML5 only (CSS coming soon)
